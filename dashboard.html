🔑 Where to Find Your Zone ID
1
Log in to galaksion.com → go to Publisher → Sites & Zones
2
Click Add Site → enter your domain → choose Pop-Under as ad type
3
After approval, open the zone. Your Zone ID appears at the top (e.g. 123456) and inside the generated script tag as ?zone=123456
4
Open index.html and replace ZONE_ID_HERE on line 20 with your actual Zone ID
5
Deploy the site → come back to this dashboard to see impressions, revenue, and visitor data
🚀 Upload to GitHub Pages (Free Hosting)
1
Create a free account at github.com → click New Repository
2
Name it anything (e.g. healthwise-site) → set it Public → click Create
3
Drag and drop all site files (index.html, css/, js/) into the repo using the browser uploader
4
Go to Settings → Pages → Source: Deploy from branch → main → / (root) → Save
5
Your site is live at https://yourusername.github.io/healthwise-site/ within ~2 minutes
6
Add this URL to Galaksion as your site URL to complete verification
